14. Registering MIST

Registration of the MIST program is voluntary and enables you to be notified when new versions are released. It also enables you to provide feedback that helps us improve the library and its documentation. To register, send an e-mail message to thomas.germer@nist.gov, indicating your interest in registering your copy of the software.

You can also follow Thomas A. Germer's GitHub site or watch the MIST GitHub page. Following or watching requires GitHub registration.
