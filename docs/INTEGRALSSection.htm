5.6 INTEGRALS Section

The INTEGRALS section defines the integrals that will be evaluated by the program. There are five different types of integrals: HEMI, CIRCLE, ANNULUS, POLYGON, and DIFFERENTIAL. Each integral must be given a unique name. To add a new integral, the user must select the INTEGRALS section, right-click with the mouse, and select Add. A dialog box will open enabling the user to choose one of the five types of integrals:
